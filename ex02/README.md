# Exercício 2 - Funcionárias com Salário Acima de R$ 5.000

## 📌 Descrição
O programa lê e armazena em vetores os dados de 5 funcionários, incluindo:
- Nome
- Sexo
- Salário

Após o cadastro, o sistema exibe apenas as funcionárias do sexo feminino que possuem salário superior a R$ 5.000.

Caso nenhuma funcionária atenda aos requisitos, uma mensagem informativa é apresentada.

## 🛠️ Tecnologias
- JavaScript
- Visual Studio Code

## ▶️ Execução
Executar o arquivo em um ambiente que suporte JavaScript e entrada de dados via `prompt`.

## 📋 Exemplo de Saída

funcionárias mulheres que ganham mais de R$5 mil.

Nome: Maria | Salário: R$ 6500.00
