# Exercício 3 - Maior e Menor Altura

## 📌 Descrição
O programa permite cadastrar o nome e a altura de várias pessoas, armazenando os dados em vetores.

O cadastro continua até que o usuário informe que não deseja adicionar novas pessoas.

Ao final, o sistema identifica e exibe:
- A maior altura cadastrada e a quem pertence.
- A menor altura cadastrada e a quem pertence.

## 🛠️ Tecnologias
- JavaScript
- Visual Studio Code

## ▶️ Execução
Executar o arquivo em um ambiente que suporte JavaScript e entrada de dados via `prompt`.

## 📋 Exemplo de Saída

Resultados

Maior altura: 1.90m e pertence a Carlos

Menor altura: 1.58m e pertence a Julia
