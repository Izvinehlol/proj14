# Exercício 1 - Maior Idade

## 📌 Descrição
O programa lê a idade de 4 pessoas e armazena os valores em um vetor.

Durante a leitura, o programa identifica qual é a maior idade informada.

Ao final, exibe:
- Todas as idades digitadas.
- A maior idade encontrada.

## 🛠️ Tecnologias
- JavaScript
- Visual Studio Code

## ▶️ Execução
Executar o arquivo em um ambiente que suporte JavaScript e entrada de dados via `prompt`.

## 📋 Exemplo de Saída

Idades digitadas: 18,22,35,27

Maior idade digitada: 35 anos.
